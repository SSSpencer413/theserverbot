PVP Commands

A list of commands that involve attacking other users.

  • duel <@User>

    Request a duel for a person. Place your bet, and (hopefully) win some credits!

  • pickpocket <@User>

    Pickpocket a person. Try not to get caught. If you're the owner, you can catch the theif for them!

Economy Commands

A list of commands for the bot's economy.

  • balance (@User)

    Shows the balance of a user.

  • pay <@User>

    Pay a person a certain amount. Currency Type could be Credits or Cubes.

  • leaderboard

    Look at the top 10 richest users in the server!

Miscellaneous Commands

Miscellaneous commands for various things! These commands will work with the default prefix in any server!

  • pvp (true/false)

    Displays if the server has pvp on or off. If you are Bot Manager+, you can set pvp.

  • commands

    Shows a list of the commands.

  • invite

    Invite the bot to your server.

  • setup

    Gives instructions on how to set the bot up if you are the owner.

  • prefix (prefix)

    Check the prefix of the server. If you are the owner, you may set the prefix.

  • info

    Get info on the shard.

  • setbotchannel

    Set the current channel to the channel where the bot will post updates and stuffs.

  • serverinfo

    Get info on the server!

  • usererinfo (@User)

    Get info on a user!
